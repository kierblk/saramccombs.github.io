Tumho Installation

INSTALLATION

Tumho is based on create-react-app and has its README.md which you can find very useful.

To get started follow this steps:

  1. Install Node and npm.
  2. Download and open project.
  3. Install packages: npm i, npm install or yarn install.
  4. Start project locally: npm start or yarn start (running on port 3014).
  5. Make necessary changes.
  6. Build app for production: npm run build or yarn build.
